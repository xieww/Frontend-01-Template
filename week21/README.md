# 毕业总结

## 时光荏苒 岁月如梭

> 从 3 月份到现在不知不觉已经五个月了，每周四、周六准时等着 winter 的直播课都成了一种习惯。就如刚开始的课程介绍一样，这几个月真没有讲什么新奇的技术，很多都是一些基础性却很容易忽视的知识。winter 教我们用溯源法去探究知识，对一个问题进行抽丝剥茧，寻根究底。

> 自己大学虽然是计算机专业，但是毕业后做的是Java，工作中虽然也负责一些前端工作，也只是停留在刀耕火种的年代。CSS基本上没写过，HTML、JavaScript也写的不多，很多就是复制粘贴修修改改而已。什么ECMA，ES6根本就没听说过，更别提React、Vue、Angular之类的前端框架以及webpack、gulp构建工具了。突然有一天意识到自己不能再这样下去，不能当一条又咸又菜的咸鱼，就网上找了一些东西开始，每天晚上下班自学。刚开始看别人都说React太难，Vue好学自己也就随大流看了一周Vue，花了两周写了个小demo，当中也出现了一些小插曲。后来裸辞赋闲了一段时间找了一份前端工作，虽然并不是很好，但也总算摸进了前端的门。从入职到现在，从自己第一次零经验独立负责项目到现在，经手的项目已经多达十余个。做的越多越感觉在吃老本，技术上也没有太多积累，不管是工作任务重还是自身的问题，事实情况就是自己在原地踏步--也许这就是瓶颈吧。自己想寻求改变，或者有人带，但是在小公司这些也没有，什么都需要靠自己。很多东西自己想尝试，可惜工作中也没有这样的业务场景，老大也不关心这些，感觉挺无力的。

> 虽然这个课程说是可以达到P6+,但是报名这个课程也不只是为了这个噱头。之所以会报名这个课程是想通过winter老师的引导去补全一些自己缺失又忽视掉的东西，夯实基础，寻求飞升。课程一开始讲了很多自己以前从来没听过的知识，基本上就是盲区，听得自己一脸懵逼，而几次课下来发现前端的水真的好深，自己现在所知所会的也只是冰山一角。后面的编程训练、工程体系也对前端有了新的认识。五个月不长也不短，虽然课程要结束了，但是自己重学前端之旅还未结束。

>总结一句话就是：知道的越多，不知道的也越多。啊，感觉时间不够用，好想再活五百年。
